## Running tests

On Linux:

```
$ make test
```

On Windows:

```
C:\src\hello-world> python -m venv venv
C:\src\hello-world> venv\Scripts\activate
C:\src\hello-world> python manage.py test
```

# Installation

```
$ limeplug install limeplugins/hello-world
```
